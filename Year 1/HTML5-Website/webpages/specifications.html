Course Specification 1

Below contains a brief explanation of the units that are studied in year 1 when studying Computer Science at Manchester Metropolitan University.

Computer Science BSc in Year 1

  1. Programming

Java is the programming language that we learn within year 1. In this this time, we gain knowledge about some of the common cross syntax practices used, like for loops, local and global variables and much more, while building on the problem-solving skills you possess.

  1. Web Design & Development

HTML5 is fifth and currently the newest major released mark-up language available. In this unit we learn to code a website using HTML5 from the ground up, using the HTML5 technologies and standards (style with CSS and make it come to life with the help of JavaScript).
